## DevSecOps CI/CD Infrastructure Project

### Purpose

This is a **learning-by-building** project focused on designing and operating a production-grade CI/CD pipeline and cloud infrastructure. While the core API application is intentionally simple, it serves as the vehicle for mastering complex delivery and operational workflows.

---

### Core Objectives

The primary goal is to practice and demonstrate proficiency in the following domains:

* **CI/CD Pipelines:** Automating the journey from code commit to deployment.
* **Containerization:** Packaging applications for consistency across environments.
* **Infrastructure as Code (IaC):** Provisioning and managing cloud resources through definition files.
* **Environment Separation:** Implementing strict boundaries between **Dev** and **Prod** environments.
* **DevSecOps Practices:** Integrating security checks (scanning, linting, and secrets management) directly into the workflow.
* **Monitoring & Operations:** Gaining visibility into system health and performance.
